# Rust Guide

欢迎来到 Rust 学习指南。本指南旨在帮助你从零开始掌握 Rust 编程语言，涵盖了从基础语法到高级并发编程的各个方面。

## 目录

### [1. 导论](1.导论.md)
介绍 Rust 的历史背景、设计目标（性能、安全、并发），以及核心概念如编译与解释、栈与堆、内存管理模式。

### [2. Rust 基础语法](2.Rust基础语法.md)
讲解变量与可变性（`let`）、数据类型、函数等基础语法知识，帮助你快速上手写出第一行 Rust 代码。

### [3. 理解所有权](3.理解所有权.md)
深入解析 Rust 最核心的特性——所有权系统，包括所有权规则、借用机制（Borrowing）以及 Slice 类型，这是理解 Rust 内存安全的关键。

### [4. 结构体和枚举](4.结构体和枚举.md)
介绍如何定义和使用结构体（Struct）和枚举（Enum），以及强大的模式匹配（Pattern Matching）功能，用于构建自定义数据类型。

### [5. 常见的集合类型](5.常见的集合类型.md)
涵盖 `Vec<T>`、`String`、`HashMap<K, V>` 等常用集合类型及其使用方法，以及迭代器的核心概念。

### [6. 包管理以及模块管理](6.包管理以及模块管理.md)
讲解 Rust 的模块系统，包括 Package、Crate、Module 的概念，以及如何使用 Cargo 组织代码和管理依赖。

### [7. 泛型 Trait 以及生命周期](7.泛型Trait以及生命周期.md)
探讨泛型编程、Trait（特征）的使用，以及生命周期（Lifetimes）的概念和应用，这是编写通用且安全代码的基础。

### [8. 面向对象以及面向函数编程](8.面向对象以及面向函数编程.md)
讨论 Rust 中的面向对象特性（封装、组合、多态）和函数式编程特性（闭包、迭代器），展示 Rust 多范式的编程风格。

### [9. 错误与异常处理](9.错误与异常处理.md)
介绍 Rust 的错误处理机制，区分可恢复错误（`Result`）和不可恢复错误（`panic!`），以及如何优雅地处理各种异常情况。

### [10. 内部可变性与智能指针](10.内部可变性与智能指针.md)
深入讲解智能指针（如 `Box`, `Rc`, `RefCell`）以及内部可变性模式，解决复杂场景下的所有权和借用问题。

### [11. Rust 中的同步并发](11.Rust中的同步并发.md)
讲解多线程编程，包括线程创建、共享内存并发（`Mutex`, `Arc`）和消息传递并发（Channel），以及 `Send` 和 `Sync` trait。

### [12. 异步编程](12.异步编程.md)
介绍 Rust 的异步编程模型，包括 `async`/`await` 语法、`Future` trait 以及 Tokio 运行时，帮助你构建高性能的异步应用。
